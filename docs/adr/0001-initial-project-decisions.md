# ADR 0001: Initial Project Decisions

## Status

Draft

## Context

This is a learning project. The user is a beginner and wants to learn a real-world software development workflow.

## Decision

Start with a simple project structure and small milestones.

Avoid advanced architecture until needed.

Use Codex as a mentor, not just a code generator.

## Consequences

Positive:

- Easier to understand.
- Easier to review.
- Easier to change later.
- Better for learning Git, testing, docs, CI/CD, and DevOps/SRE basics.

Negative:

- Some production-grade features will be added later, not immediately.
- Progress may feel slower, but learning quality should be higher.
